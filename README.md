# unbiased_cnf
Repository for unbiased formulation of CNF using stochastic dynamics. 

Create a new python environment (optional you can do the same operation using `conda`)
```bash
$ python3 -m venv .venv
```
start your virtual environment
```bash
$ source .venv/bin/activate
```
install the package via `pip`
```bash
$ pip install -e .
```
Add to the file pyproject.toml all the needed packages.
